
# 1️⃣ Первичное тестирование (базовая проверка)

## Подключение
- Подключить к разным ПК
- Подключить к ноутбуку
- Проверить работу без драйверов
- Проверить с установленным ПО

## Поверхности
- Работает ли без коврика?
- Работает ли на столе?
- Работает ли на стекле?
- Работает ли на ткани?

## Условия эксплуатации
- Работает ли при тряске (например, в поезде)?
- Работает ли зимой на холоде?
- Перегревается ли летом?

---

# 2️⃣ Аппаратная проверка

## Тип подключения
- Проводная или беспроводная?
- Тип интерфейса (USB / Bluetooth / 2.4 GHz)

## Кабель (для проводной)
- Длина кабеля
- Прочность кабеля
- Сломается ли при резком рывке?

## Корпус
- Размер и вес
- Удобно ли лежит в руке?
- Есть ли люфт кнопок?
- Колесо прокрутки не заедает?

## Механические тесты
- Можно ли уронить?
- Постучать по столу?
- Попытаться сжать?
- Что будет, если пролить воду?
- Что будет, если пыль попадёт внутрь?

## Состояние
- Новая или после длительного использования?

---

# 3️⃣ Black Box Testing (Тестирование «черного ящика»)

Проверка без анализа внутреннего устройства.

- Двигается ли курсор корректно?
- Работают ли левая и правая кнопки?
- Работает ли колесо прокрутки?
- Работают ли дополнительные кнопки?
- Работает ли на разных поверхностях?

---

# 4️⃣ Automated Testing (Автоматизированное тестирование)

- Выдержит ли 1 000 000 кликов?
- Проверка задержки отклика (latency)
- Автоматическое тестирование DPI
- Тест устойчивости сигнала (Bluetooth / 2.4 GHz)
- Автоматический тест износа колеса прокрутки

---

# 5️⃣ Functional Testing (Функциональное тестирование)

- Точный ли сенсор?
- Корректно ли работает смена DPI?
- Работает ли подсветка (если есть)?
- Работает ли ПО для настройки?
- Работают ли макросы?

---

# 6️⃣ Validation (Валидация)

Проверка соответствия требованиям.

- Соответствует ли заявленному DPI?
- Материал корпуса безопасен?
- Нет ли острых краёв?
- Кабель изолирован?
- Батарея не перегревается?
- Есть ли защита от короткого замыкания?

---

# 7️⃣ Verification (Верификация)

Проверка соответствия документации.

- Соответствует ли цвет и дизайн макету?
- Соответствует ли вес заявленному?
- Правильная ли комплектация?
- Корректная ли маркировка?

---

# 8️⃣ Usability Testing (Тестирование удобства)

- Удобно ли держать длительное время?
- Не устает ли рука?
- Подходит ли для разных типов хвата?
- Удобно ли нажимаются боковые кнопки?
- Хорошо ли скользит по столу?

---

# 9️⃣ Compatibility Testing (Совместимость)

- Работает ли с Windows?
- Работает ли с macOS?
- Работает ли с Linux?
- Работает ли с игровыми консолями?
- Совместима ли с разными ковриками?

---

# 🔟 Performance Testing (Производительность)

- Время отклика
- Поведение при высокой скорости движения
- Потеря сигнала на расстоянии (для беспроводной)
- Количество циклов клика до износа
- Износ ножек

---

# ✅ Итог

После прохождения всех этапов тестирования можно сделать вывод:
- Подходит ли мышь для офисной работы
- Подходит ли для игр
- Подходит ли для длительного использования
- Соответствует ли заявленным характеристикам
