## 🎯 Цель работы
Оценить качество работы жесткого диска с помощью специализированного программного обеспечения (CrystalDiskMark и CrystalDiskInfo).

**CrystalDiskMark** — утилита для измерения скорости чтения и записи на дисках. Позволяет тестировать разные условия (размер блоков, очередь запросов, многопоточность).

**CrystalDiskInfo** — инструмент для диагностики, считывающий данные S.M.A.R.T. (Self-Monitoring, Analysis and Reporting Technology). Показывает температуру, количество проблемных секторов, время работы и общее состояние накопителя.

## ❓ Контрольные вопросы

### 1. Что такое жесткий диск?

**Жесткий диск (HDD)** — энергонезависимое устройство хранения данных на основе принципа магнитной записи. Информация записывается на вращающиеся пластины, покрытые ферромагнитным слоем. В современном контексте термин также может относиться к SSD, хотя принципы работы кардинально отличаются.

### 2. Внутреннее устройство HDD

```
┌─────────────────────────────────────┐
│   ╔═══════════════════════════════╗ │
│   ║  ┌───┐          ╔═══╗         ║ │
│   ║  │   │  ╔═══╗   ║   ║  ╔═══╗   ║ │
│   ║  │   │  ║   ║   ║   ║  ║   ║   ║ │
│   ║  └───┘  ╚═══╝   ╚═══╝  ╚═══╝   ║ │
│   ║   Магнитные пластины (диски)    ║ │
│   ╚═══════════════════════════════╝ │
│  ┌─────────────────────────────┐    │
│  │      Блок магнитных головок │    │
│  └─────────────────────────────┘    │
│  ┌─────────────────────────────┐    │
│  │      Шпиндельный двигатель  │    │
│  └─────────────────────────────┘    │
│  ┌─────────────────────────────┐    │
│  │      Плата контроллера      │    │
│  └─────────────────────────────┘    │
└─────────────────────────────────────┘
```

**Основные компоненты:**
- **Гермоблок** — защищенный корпус
- **Магнитные пластины** — носители информации
- **Шпиндельный двигатель** — вращает пластины
- **Блок головок** — чтение/запись данных
- **Позиционер** — перемещает головки
- **Плата контроллера** — управление

### 3. Причины проблем совместимости

| Категория | Проблемы |
|:----------|:---------|
| **Интерфейс** | Несоответствие разъемов (SATA/IDE), старые кабели |
| **ПО** | Устаревший BIOS, отсутствие драйверов чипсета |
| **Питание** | Слабый БП, некачественные переходники |
| **ОС** | Отсутствие TRIM для SSD, неподдержка GPT |

### 4. Методы решения проблем совместимости

```bash
# 1. Обновление ПО
  • BIOS/UEFI до последней версии
  • Драйверы чипсета с сайта производителя

# 2. Правильный выбор железа
  • Качественные кабели SATA 3
  • Проверка QVL материнской платы
  • Достаточный запас мощности БП

# 3. Настройка ОС
  • Включение TRIM для SSD
  • Правильный выбор MBR/GPT
  • Актуальная файловая система
```
